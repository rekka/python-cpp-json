Example of running a C++ binary from Python 3 with data passed in between
via JSON files.

Uses [JSON for Modern C++](https://github.com/nlohmann/json) library.

C++ code computes the hypotenuse of a right-angled triangle.

## Requirements

 - Python 3
 - numpy, matplotlib
 - GCC

## Usage

```bash
./build
python plot.py
```
